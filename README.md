Проект "Solar Tales" - Информационно-справочная консольная система направленная на работу и обработку информации космических тел.

Обьекты: Планеты и Спутники.

Функции системы: 
1) Добавление данных.
2) Просмотр данных.
3) Удаление данных.
4) Поиск данных.
5) Сортировка данных.
6) Сохранение данных и выход из системы.

Проект находится в разработке еще неделю.
